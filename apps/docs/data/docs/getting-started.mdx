# Getting started
Learn how to get started with JHQ UI in your project.

### Prerequisites
- Next.js project
- Chakra UI set up in your `_app.tsx`
- Auth0 set up (optional)

### Installation
Inside your project's directory, install:
```
yarn add jhq-ui
```
or with npm
```
npm i jhq-ui
```

### Quick start
To see an example of how to use JHQ UI in your Next.js project, head over to our [playground](/playground).
