# Layout
A layout component that provides a header, combined with reactive data fetched from the JacobHQ API.

### Usage
<Play imports={['Layout']} content={`<Layout appTitle="Components" title="Layout" description="This is a demo of the Layout component.">Some example content</Layout>`} />

### Props
- `appTitle` - Sets the title of the app.
- `description` - The text displayed below the title.
- `title` - Sets the page title, and adds a breadcrumb if set. Falls back to `appTitle` if undefined.
- `children` - Content of the page.
- `router` - Return value from the `useRouter` hook in Next.js.
- `user` - Return value from the `useUser` hook with Auth0.
- `authLoading` - Wether you are still fetching the user or not.
